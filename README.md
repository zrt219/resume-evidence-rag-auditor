# Resume Evidence RAG Auditor

This repository is the public-facing landing page for a resume-proof auditing workflow.

## What this project is for

The goal is to inspect resume claims against local evidence and keep the final output honest, reviewable, and employer-safe.

## What this project should demonstrate

- evidence collection and claim grading
- truth-bound resume editing
- downgrade/remove flows for weak claims
- missing-proof tracking
- recruiter-safe wording and artifact hygiene

## Employer-facing framing

A reviewer should come away with one clear message: this system is designed to prevent unsupported portfolio claims and to keep resume language aligned with verifiable source material.

## Status

- Public repo scaffold: yes
- Employer-facing README: yes
- Production status: not claimed unless separately verified

## Good next additions

- example evidence ledger
- claim grading rubric
- sample audit output
- input/output examples with redacted data
